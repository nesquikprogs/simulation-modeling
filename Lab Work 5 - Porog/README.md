# Lab Work 5 - Porog (Модель пороговой активации вершин)

---

## Цель лабораторной работы

Цель работы — исследовать **модель пороговой активации вершин графа** и изучить:  

- динамику распространения активации по сети;  
- влияние порога активации на скорость и масштаб процесса;  
- визуализацию процесса в виде анимации;  
- взаимодействие структуры графа и поведения системы.  

---

## Принцип работы модели

1. **Граф:** случайный граф с `num_nodes` вершинами  
2. **Порог активации:** `threshold` — минимальное количество активных соседей для активации вершины  
3. **Изначально активные вершины:** список `initial_active`  
4. **Тип графа:** ориентированный (`directed=True`) или неориентированный (`directed=False`)  
5. **Процесс активации:**  
   - На каждом шаге вершина активируется, если число активных соседей ≥ порог  
   - Модель продолжается до стабилизации (нет новых активаций)
