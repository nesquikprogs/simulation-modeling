# Lab Work 4 - CellsAuto (Клеточный автомат)\

---

## Цель лабораторной работы

Цель работы — изучить **распространение эпидемии** с помощью клеточного автомата и исследовать:  

- динамику заражения и выздоровления;  
- влияние соседей на вероятность заражения;  
- визуализацию процесса на сетке клеток;  
- основные принципы моделирования эпидемий.  

---

## Принцип работы модели

Модель реализует **клеточный автомат с окрестностью Мура**:

1. **Состояния клеток:**  
   - `SUSCEPTIBLE` — здоров  
   - `INFECTED` — заражён  
   - `RECOVERED` — выздоровел  

2. **Алгоритм работы:**  
   - На каждом шаге клетки проверяют соседей (8 соседей — окрестность Мура)  
   - Здоровая клетка может заразиться, если есть инфицированные соседи  
   - Заражённые клетки считаются выздоровевшими после `RECOVERY_TIME` шагов  
   - Выздоровевшие клетки больше не заражаются
