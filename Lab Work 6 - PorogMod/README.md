# Lab Work 6 - PorogMod (Модифицированная модель пороговой активации)

---

## Цель лабораторной работы

Цель работы — исследовать **модифицированную модель пороговой активации вершин графа**, где активация зависит от веса связей, и изучить:  

- влияние веса ребер на распространение активации;  
- динамику активации в сети с учетом средней силы влияния соседей;  
- визуализацию процесса через анимацию;  
- различие поведения между простой и взвешенной моделью.  

---

## Принцип работы модели

1. **Граф:** случайный граф с `num_nodes` вершинами  
2. **Вес ребер:** случайные значения от 0 до 1  
3. **Порог активации:** `threshold` — среднее влияние соседей, необходимое для активации вершины  
4. **Изначально активные вершины:** `initial_active`  
5. **Тип графа:** ориентированный (`directed=True`) или неориентированный (`directed=False`)  
6. **Процесс активации:**  
   - На каждом шаге вычисляется среднее влияние соседей для каждой неактивной вершины  
   - Если среднее влияние > порог, вершина активируется  
   - Модель продолжается до стабилизации (нет новых активаций)
