# Simulation Modeling - Lab Works (6 semester)

---

В этом репозитории собраны лабораторные работы по курсу "Имитационное моделирование" за 6 семестр. Все работы выполнены на **Python** с использованием различных библиотек (`matplotlib`, `networkx`, `tkinter`, `scipy`) и ориентированы на изучение методов имитационного моделирования:  

- системная динамика (iThink),  
- системы массового обслуживания (СМО),  
- цепи Маркова,  
- клеточные автоматы,  
- пороговые модели активации,  
- эпидемические модели SIS/SIR.
