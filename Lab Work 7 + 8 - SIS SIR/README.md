# Lab Work 7 + 8 - SIS / SIR Models

---

## Цель лабораторной работы

Цель работы — исследовать **модели распространения инфекционных заболеваний на графах**:  

1. **SIR Model** — инфекция с выздоровлением и иммунитетом  
2. **SIS Model** — инфекция с выздоровлением без иммунитета  

Задачи:  

- изучить динамику заражения и выздоровления в сети;  
- исследовать влияние вероятности заражения и выздоровления на масштаб эпидемии;  
- визуализировать процесс через анимацию;  
- сравнить поведение двух моделей.  

---

## Принцип работы моделей

### 1. SIR Model (`sir.py`)

- **Состояния вершин:**  
  - `S` — восприимчивый (susceptible)  
  - `I` — инфицированный (infected)  
  - `R` — выздоровевший (recovered)  

- **Правила:**  
  1. Восприимчивые вершины заражаются с вероятностью `p`, если сосед инфицирован  
  2. Инфицированные вершины выздоравливают с вероятностью `q` и становятся `R`  
  3. Вершины `R` остаются невосприимчивыми  

### 2. SIS Model (`sis.py`)

- **Состояния вершин:**  
  - `S` — восприимчивый  
  - `I` — инфицированный  

- **Правила:**  
  1. Восприимчивые вершины заражаются с вероятностью `p` от каждого инфицированного соседа  
  2. Инфицированные вершины выздоравливают с вероятностью `q` и снова становятся `S`
