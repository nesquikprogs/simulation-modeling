# Lab Work 3 - CepMark (Марковские цепи)
---

## Цель лабораторной работы

Цель работы — изучить **Марковские цепи с поглощением** и исследовать их поведение через имитационное моделирование.  
Лабораторная позволяет:  

- моделировать траектории цепи с конечным количеством состояний;  
- анализировать время до поглощения;  
- строить статистику финальных состояний;  
- визуализировать траектории и матрицу переходов.  

---

## Принцип работы модели

1. **Состояния цепи:** N_STATES = 30  
2. **Поглощающее состояние:** ABSORBING_STATE = 29  
3. **Траектории:** N_TRAJECTORIES = 1000  
4. **Максимальная длина траектории:** MAX_STEPS = 200  
5. **Генерация матрицы переходов:** случайные вероятности с корректировкой для поглощения  
6. **Симуляция:** случайное прохождение цепи до поглощения или максимального числа шагов  
7. **Сбор статистики:**  
   - время до поглощения;  
   - количество траекторий, достигших поглощения;  
   - распределение финальных состояний;  
   - визуализация графиков.
